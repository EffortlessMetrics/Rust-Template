# Backstage Documentation

This directory contains documentation and resources for the Backstage integration with the Rust-as-Spec platform cell.

## Overview

Backstage serves as the developer portal for the Rust template, providing centralized access to:

- API documentation
- Component discovery
- Developer workflows
- Governance artifacts
- Integration with CI/CD systems
